# What this is about?
This is a project I made while studying the [Crafting Interpreters](https://craftinginterpreters.com/) book, by Robert Nystrom.

In this project, I wrote not only the code provided by the book, but also the proposed challenges and many personal notes/insights I had while developing the interpreter and exploring the uncharted lands my curiosity drove me to.

Since a part of the interpreter development is made in C, this was also a perfect opportunity to strengthen my knowledge in this fascinating language.

# A bit of context
While in my college course of System Analysis and Development, I had few opportunities do study and learn more about C programming, data structures, code optimization and low level development, since this course is more focused on the preparation for the job market and professional development.

But, somehow, the few classes I had about these subjects, sparked me a fierceful curiosity and delightful times trying to solve a seg fault issue on assignments. This isanity droves me more and more towards this "deep dark secret" side of technology. 

After getting my degree, I started to pursue this kind of knowledge further and further, until I discovered about compilers and interpreters. A curious union of these "deep into the machine" knowledge that I've been so eager to work with plus one of my most toiful hobby: learning new languages.
